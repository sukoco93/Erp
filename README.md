# ERP Tempe

Aplikasi manajemen konsinyasi berbasis Vue 3 + Dexie + Tailwind.

## Fitur
- Manajemen pelanggan, barang, konsinyasi
- Pembayaran, retur, cicilan
- Audit log, filter tanggal, undo/redo
- PWA-ready, mobile-first

## Cara Menjalankan
Buka `index.html` dengan Live Server atau deploy ke GitHub Pages.
